# RealSense Camera Tracking

## Setup on Windows PC

1. Clone this repo:
```bash
git clone <your-repo-url>
cd camera_track
```

2. Create virtual environment and install dependencies:
```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

3. Run the code:
```bash
python test.py
```

## Workflow

**On Mac:** Code and push changes
```bash
git add .
git commit -m "your changes"
git push
```

**On Windows:** Pull and run
```bash
git pull
python test.py
```
